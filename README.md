# Michael Ivan Santoso Portfolio

Personal portfolio website for showcasing projects, profile information, and resume.

## Tech Stack

- React
- TypeScript
- Vite
- Chakra UI
- Framer Motion
- React Router

## Pages

- Home
- Projects
- About
- Resume / CV

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Project Structure

```text
public/      Static assets such as images and PDF files
src/         React source code
src/pages/   Main pages
src/components/ Reusable UI components
src/data/    Portfolio project data
```
